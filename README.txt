Essa aplicação web simula o cotidiano de um negócio 
local, onde é estabelecida uma meta diária. A cada venda, 
o usuário insere o valor na aplicação, que automaticamente 
calcula e exibe quanto falta para atingir a meta ou se ela 
já foi alcançada. Se a meta for ultrapassada, a aplicação 
informa o valor excedente como "lucro". O código JavaScript 
contém funções que capturam os valores inseridos, realizam 
os cálculos necessários e atualizam dinamicamente os espaços 
correspondentes na página. Além disso, há uma função que mantém 
a data atualizada, eliminando a necessidade do usuário inserir a 
data a cada visita.

A estilização da página foi implementada utilizando apenas CSS, 
incluindo um design responsivo para telas com largura máxima de 
600 pixels. Isso garante uma experiência visual otimizada em 
dispositivos móveis ou em janelas de navegador mais estreitas. 
A interface gráfica busca ser intuitiva e amigável, proporcionando 
uma interação fácil e eficaz para os usuários.
